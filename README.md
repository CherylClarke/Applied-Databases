Applied Databases

Main.py Project

This project was developed as part of the Applied Databases module for the Higher Diploma in Science in Data Analytics. The application is a conference management system built in Python using both MySQL and Neo4j databases. The system allows users to view conference sessions, attendees, rooms, and attendee connections, as well as add new attendees and attendee relationships. The project demonstrates database connectivity, SQL queries, graph database queries, input validation, error handling, and menu-driven application development using Python.
